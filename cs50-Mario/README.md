# Mario
## Terminal pyramids

* Program that prints pyramids in the terminal.

### Conditions:
* Height from 1 and 8.
* 2 spaces in between.
* The program validates input user like letters/words, decimals, numbers  
bigger than 8 or smaller than 1.

### Examples

* Height 1:    
'''
#  #
'''

* Height 3:  
'''
   #  #
  ##  ##
 ###  ###
 '''
